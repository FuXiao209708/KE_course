# 快速启动指南

## 启动步骤

### 1. 启动后端服务器

```bash
cd "f:\SEU_projects\2605知识工程课视频\demo"
python -m uvicorn backend.app:app --host 127.0.0.1 --port 8000
```

或者使用conda环境：
```bash
cd "f:\SEU_projects\2605知识工程课视频\demo"
"/c/Users/lrf/.conda/envs/cdkg/python.exe" -m uvicorn backend.app:app --host 127.0.0.1 --port 8000
```

### 2. 访问系统

打开浏览器，访问：**http://127.0.0.1:8000**

### 3. 查看班级学情分析

1. 在首页点击"班级学情分析"卡片，或
2. 点击左侧边栏的"班级学情分析"按钮

## 技术栈

- **后端**: Python + Starlette + Uvicorn
- **前端**: HTML5 + Tailwind CSS + 原生JavaScript
- **可视化**: 自定义SVG图表
- **数据**: 模拟数据（用于演示）

## 注意事项

1. 确保Python环境已安装所需依赖：
   ```bash
   pip install starlette uvicorn
   ```

2. 服务器启动后，终端会显示：
   ```
   INFO:     Uvicorn running on http://127.0.0.1:8000
   ```

3. 当前使用模拟数据，实际部署时需接入真实数据源

## 故障排查

### 问题：服务器无法启动
- 检查端口8000是否被占用
- 确认Python环境和依赖是否正确安装

### 问题：页面显示异常
- 清除浏览器缓存
- 检查浏览器控制台是否有JavaScript错误
- 确认所有静态资源路径正确

### 问题：图表不显示
- 检查JavaScript控制台错误
- 确认analytics_handler.js已正确加载
- 验证数据格式是否正确
